get /transaction/{reponame}/{itemname}/{tag}

【拥有者】校验该access_token的有效性，此access_token被需求者的私有证书加密 注意：本组api接口只有这个不需要验证用户身份。 注意：tag需要UTF8编码

  • Request
  • Response

URI Parameters

  • reponame: required (string)

    repository名称

    Example:

    Intenet_stats
  • itemname: required (string)

    dataitem名称

    Example:

    Cars
  • tag: required (string)

    tag名称

    Example:

    Cars_yieche_20151101

Query Parameters

  • cypt_accesstoken: required (string)

    需求者的access token

    Example:

    cypt_accesstoken=4460f5708fdc53fc
  • username: required (string)

    需求者用户名

    Example:

    username=shendf

HTTP status code 200

Body

Type: application/json

Schema:

valid：输入access token是否有效
remainingtime：剩余有效期。可能的结果："33h3m0.5s", "3m2s", "0"

Example:

{
  "code": 0,
  "msg": "OK",
  "data": {
    "valid":true
    "remainingtime":"33h3m0.5s"
  }
